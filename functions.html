Tower Defense  0.1
Ce projet dans le cadre de notre formation de L2 informatique à l'Université Du Mans a pour but de créer un jeu de type "Tower Defense" en utilisant la librairie graphique SDL2.
Liste de tous les champs de structure et d'union documentés avec des liens vers la documentation de structure/union de chaque champ :
  • argent : joueur_S
  • case_pl : case_S
  • cases : map_S
  • chemin : case_U
  • cout : tour_S
  • degat : ennemi_S , tour_S
  • emplacement : case_U
  • enemi : chemin_S
  • frequence : tour_S
  • id : ennemi_S , tour_S
  • portee : tour_S
  • position : chemin_S , emplacement_S , ennemi_S , tour_S
  • suivant : chemin_S
  • tour : emplacement_S
  • type : case_S
  • vie : ennemi_S , joueur_S
  • vitesse : ennemi_S
  • x : position_S
  • y : position_S
